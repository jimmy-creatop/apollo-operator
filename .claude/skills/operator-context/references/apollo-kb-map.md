# Apollo knowledge base map

> Where to send the human when the MCP hits its edge. The Apollo MCP automates data, enrichment, sequences, sending, tasks, and analytics. Everything else (personas, saved-search alerts, workflows, the dialer, meetings, waterfall enrichment, deals) lives in the Apollo UI. When a skill needs one of those, point the human to the exact article here and tell them what to do. Do not pretend the MCP can do it.
>
> URLs are public Apollo docs and Academy resources. Organized by the level each supports.
